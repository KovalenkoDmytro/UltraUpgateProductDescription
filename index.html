Korzystaj z dwóch systemów jednocześnie! MacOS & Windows

Pracując na macOS można uruchomić pełnoprawny Windows 8,9 lub 11. Otrzymasz nie tylko elegancję i wydajność, ale także możliwość korzystania z systemu Windows!

  1. Dynamiczna zmiana rozdzielczości Windows'a podczas zmiany rozmiaru okna, w którym znajduje się zwirtualizowany system lub w przypadku przełączenia trybu wyświetlania na pełny ekran
  2. Współdzielenie plików pomiędzy systemami za pomocą przenoszenia ich do specjalnego folderu służącego lub przeciąganie i upuszczanie ich na okno w którym wyświetlany jest Windows
  3. Współdzielenie schowka (kopiowanie i wklejanie tekstu)
  4. Możliwość "podpięcia" dowolnego urządzenia USB pod wirtualny system Windows
  5. Możliwość wyświetlania okienek programów działających w systemie Windows tak jakby były natywnymi programami pracującymi w systemie macOS
  6. Możliwość przeglądania plików znajdujących się w Windowsie z poziomu macOS i odwrotnie (brak potrzeby przenoszenia plików do specjalnego folderu, aby były widoczne dla obu systemów jednocześnie)

Boot Camp to narzędzie dzięki któremu tuż przy starcie komputera wybieramy z którym systemem chcemy uruchomić komputer. Do wyboru jest Applowski system Mac OS lub Microsoft Windows.

Gdy wybierzemy system Windows, iMac uruchomi się wyłączenie z systemem Windows. Co daje nam taka opcja? iMac staje się wtedy "pecetem" Windows ma pełny dostęp do podzespołów komputera. W pełni korzysta z procesora, pamięci RAM, czy też karty graficznej. Możemy więc korzystać z pełnej wydajności w programach graficznych, muzycznych czy do edycji video, a które to działają tylko w systemie Windows. Możemy także grać w ulubione gry, które wymagają systemu Windows.
